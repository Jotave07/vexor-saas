# VEXOR Sistemas

Projeto frontend da VEXOR Sistemas para operacao de e-commerce e painel administrativo.

## Banco de dados

O projeto original foi gerado com integracao Supabase. Nesta etapa foi adicionada a base da migracao para MariaDB em `database/mariadb/schema.sql`.

Para concluir a conexao com a Hostinger ainda falta informar o host do banco.
